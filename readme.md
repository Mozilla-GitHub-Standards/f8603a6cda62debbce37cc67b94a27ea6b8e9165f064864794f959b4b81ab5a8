# New new tab

## To run

```
npm install
npm start
```

Set your remote new tab to point to `http://localhost:1944`.

## What's working?

* Search suggestions
* Prefs are loaded (but not settable)
* Fetching directory/suggested tiles
* Fetching history tiles
